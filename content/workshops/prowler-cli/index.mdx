This workshop walks you through installing Prowler CLI, running your first security scan against AWS, exporting reports in multiple formats, and exploring findings through the interactive dashboard.

Steps are provided for Linux, Windows, and macOS side by side. Follow the instructions that match your machine.

## What You'll Learn

- Installing Prowler CLI via pip
- Running a full AWS security scan
- Scoping scans by service and severity
- Exporting findings as CSV, JSON-OCSF, and HTML
- Using the Prowler Dashboard to explore results
- Deploying a workshop EC2 environment with CloudFormation
