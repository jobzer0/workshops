## Default Output Files

Prowler writes output to an `output/` folder inside your current directory. The following formats are generated by default:

| Format | File | Purpose |
|--------|------|---------|
| CSV | `output/*.csv` | Machine-readable findings (semicolon-delimited) |
| JSON-OCSF | `output/*.ocsf.json` | Open Cybersecurity Schema Framework format |
| HTML | `output/*.html` | Human-readable report you can open in a browser |

## View Your Results

| Linux | Windows | macOS |
|-------|---------|-------|
| `ls -la ~/Documents/output/` | `dir %USERPROFILE%\Documents\output\` | `ls -la ~/Documents/output/` |
| Open HTML: `xdg-open ~/Documents/output/*.html` | Open HTML: double-click the `.html` file in Explorer | Open HTML: `open ~/Documents/output/*.html` |

## Control Output Formats

Specify which formats to generate explicitly:

```bash
prowler aws --output-formats csv json-ocsf html
```

## Custom Output Directory

Send output to a specific directory instead of the default `./output/` folder:

| Linux | Windows | macOS |
|-------|---------|-------|
| `prowler aws --output-directory ~/Documents/prowler-reports` | `prowler aws --output-directory %USERPROFILE%\Documents\prowler-reports` | `prowler aws --output-directory ~/Documents/prowler-reports` |
